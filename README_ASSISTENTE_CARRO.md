# 🚗 Assistente Carro - Inteligência Artificial em Português Brasileiro

Um assistente de IA conversacional especializado para motoristas, com suporte completo a reconhecimento de voz, síntese de fala em português brasileiro e interface intuitiva para uso em veículos.

## ✨ Características Principais

### 🎤 Reconhecimento de Voz
- Captura contínua de áudio em português brasileiro
- Reconhecimento em tempo real com feedback visual
- Suporte para múltiplos sotaques e dialetos

### 🔊 Síntese de Fala (TTS)
- Leitura automática de respostas em português brasileiro
- Sincronização perfeita entre texto e áudio
- Qualidade de voz natural e fluida

### 🧠 Inteligência Artificial
- Processamento de linguagem natural avançado
- Respostas contextualizadas e relevantes
- Modo turbo para respostas rápidas

### 📊 Análise em Tempo Real
- Monitoramento de nível de microfone
- Análise de sentimento e emoção
- Histórico completo de conversas

### 🎨 Interface Responsiva
- Design otimizado para mobile e desktop
- Modo escuro/claro automático
- Controles de volume integrados

## 🚀 Como Usar

### Acesso Rápido
1. Abra a página do Assistente Carro
2. Clique no botão de microfone para começar
3. Faça sua pergunta em voz alta
4. Aguarde a resposta em áudio

### Funcionalidades
- **Modo Turbo:** Respostas mais curtas e rápidas
- **Histórico:** Acesso a conversas anteriores
- **Controle de Volume:** Ajuste o nível de áudio
- **Mudo:** Desative o áudio quando necessário

## 🛠️ Tecnologias

- **Frontend:** React 19 + TypeScript + Tailwind CSS
- **Backend:** Node.js + Express + tRPC
- **Banco de Dados:** MySQL com Drizzle ORM
- **Hospedagem:** Vercel (recomendado)
- **APIs:** Web Speech API, OpenAI, Google Cloud

## 📦 Instalação Local

```bash
# Clonar repositório
git clone https://github.com/tafita81/Dani.git
cd Dani

# Instalar dependências
pnpm install

# Configurar variáveis de ambiente
cp .env.example .env.local

# Build
pnpm build

# Iniciar servidor
pnpm start
```
